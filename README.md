Description:
Developed a feature-rich Netflix clone using modern technologies, including React, Node.js, Express, MongoDB, Firebase, and JWT authentication. This platform enables users to enjoy a seamless movie streaming experience while providing comprehensive administrative functionalities.

Highlights:

Created a responsive frontend interface using React, delivering a user-friendly and visually appealing design.
Implemented user registration and authentication using JWT tokens, enhancing security and ensuring secure access to the platform.
Utilized Firebase cloud storage to efficiently upload, manage, and serve images and videos for an immersive viewing experience.
Established a robust backend using Node.js and Express, facilitating efficient data management and manipulation.
Designed an intuitive admin panel with capabilities to manage movies, users, and movie lists (genres).
Enabled admins to add, edit, and delete movies, users, and lists through the admin panel.
Leveraged MongoDB Atlas to store and manage project data, ensuring data integrity and scalability.
Hosted the backend on Render.com and the frontend on Netlify, ensuring high availability and seamless performance.
Skills Utilized:
React, Node.js, Express, MongoDB, Firebase, JWT Authentication, REST API, Admin Panel Development, Database Management, Cloud Storage, Responsive Design.
